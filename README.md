## 🚀 Getting Started

### View the Site
1. Open `index.html` in your web browser
2. All styling is automatically applied
3. Test on different screen sizes

### Customize the Design

**Change Colors**:
```css
/* In css/styles.css, modify :root variables */
:root {
    --color-primary: #6F4E37;  /* Change this to your color */
    --color-accent: #D4A574;   /* Change accents */
}
```

**Adjust Typography**:
```css
:root {
    --font-size-base: 1rem;    /* Adjust base size */
    --font-size-3xl: 2.5rem;   /* Adjust heading size */
}
```

**Modify Spacing**:
```css
:root {
    --spacing-md: 1rem;    /* Adjust spacing */
    --spacing-lg: 1.5rem;  /* All elements scale */
}
```
